# Applauz — Sprint 1 : Base 44

Objectif : concentrer Applauz sur la Loire-Atlantique avant d'étendre aux autres départements.

Ce sprint ajoute :

- une table de suivi des communes du 44 ;
- un référentiel de lieux canoniques ;
- une table d'alias pour regrouper les variantes de noms ;
- une table de propositions de correspondance entre ton agenda et les lieux ;
- des points de qualité à corriger ;
- une interface plus lisible pour consulter la base en construction.

## Commande à lancer

Depuis la racine du projet :

```bash
python scripts/sprint1_upgrade_44.py
```

Puis lancer Applauz :

```bash
python app/app.py
```
