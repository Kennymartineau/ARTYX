COMMANDES A LANCER DANS LE DOSSIER APPLAUZ SUR WINDOWS

1) Ouvrir le dossier dans VS Code
2) Ouvrir le terminal dans VS Code
3) Copier-coller :

git init
git add .
git commit -m "Initial commit APPLAUZ"
git branch -M main
git remote add origin https://github.com/TON-UTILISATEUR/APPLAUZ.git
git push -u origin main

Remplace TON-UTILISATEUR par ton nom d'utilisateur GitHub.
